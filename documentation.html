EECS 448 Project #1: Clock display.

A project for EECS 448 - Software Engineering I at the University of Kansas. The web-based clock has the functions of setting up the time and changing between standard hour and military hour. On Github

Authors:
- Luke Dercher
- Sri Gayatri
- Luke Weber
- Vuong Nguyen.
Date: Feb 14, 2016.
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Features:
- The users can set up the time to start with.
- The clock can change between standard hour (am/pm) and military/24 hour.
- The clock can detects invalid inputs and flashes 12:00:00.

Pre and post conditions of the code functions:

function init_blink(MaxHour, MinHour)
-Precondition: Must have a valid MaxHour and MinHour to pass to startClock() to start the clock back up again when valid input is entered
-Postcondition: Calls the blink method over and over again until correct input is given
-Authors: Luke Dercher, Sri Gayatri, Luke Weber.

function blink(MaxHour, MinHour)
-Precondition: Must have a valid MaxHour and MinHour to pass to startClock() to start the clock back up again when valid input is entered
-Purpose:changes the color of the 1200 text from black and white
-Postcondition: exits the function when counter is set to stop
-Authors: Luke Dercher, Sri Gayatri, Luke Weber.

function startClock(MaxHour, MinHour)
-Precondition: Must have a valid MaxHour and MinHour to decide whether to display 24 hr or 12 hr time
-Purpose: starts the clock. Called recursively. Adds one second every time called and then increments minutes and hours
-Postcondition: Must display inputted time and change to am or pm when nesseccary
- Author: Sri Gayatri & Luke Weber.

function checkTime(i)
-Precondition: feeds in an int
-Purpose: adds a zero in front of numbers when needed
-Postcondition:
-Author: Sri Gayatri.

function clockBuild()
-Precondition:
-Purpose: checks for good input
-Postcondition: send that input to startClock()
-Author: Luke Dercher, Sri, and Luke Weber.

function chngDisp()
-Precondition: must read in value from 12hr or 24 hr drop box
-Purpose: change display of what kind of input is acceptable
-Postcondition:
-Author: Luke Dercher and Luke Weber.

Class Clock()
-Precondition: none
-Purpose: A static object to store all variables of a clock
-Postcondition: Creates clock "object."
-Authors: Luke Weber.
